## HTTP request / response

Use the `http` fence (aliases: `request`, `response`, `req`, `res`). Bodies that look like JSON get nested highlighting at runtime.

```http
POST /api/v1/login HTTP/1.1
Host: target.example
User-Agent: Mozilla/5.0
Content-Type: application/json
Authorization: Bearer eyJhbGciOiJIUzI1NiJ9.example
Cookie: session=abc123

{
    "username": "admin",
    "password": "Winter2024!"
}
```

```http
HTTP/1.1 200 OK
Server: nginx/1.22.1
Content-Type: application/json
Set-Cookie: session=pwned; Path=/; HttpOnly

{
    "success": true,
    "role": "admin"
}
```

Plain `text` fences that *start* like HTTP are auto-promoted to `language-http` by the converter:

```text
GET /api/v1/user?email=nj@tvhsecurity.com HTTP/2
Host: target.example
Accept: application/json
```
